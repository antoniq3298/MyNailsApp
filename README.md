# My Nails

My Nails is a mobile application designed for nail artists and clients.  
It allows users to browse a gallery of nail designs, mark favorites, and book appointments.

## Features
- 📸 Gallery of nail designs
- ❤️ Add designs to Favorites
- 📝 Description for each design
- 📅 Appointment booking
- Offline mode support

## Technologies Used
- Java / Kotlin
- Android Studio
- Glide (for image loading)
- Material Design UI
